# 📱 pamiii-cristian-harold

Repositório da disciplina **Programação de Aplicativos Mobile III**, ministrada pelo Professor João Siles.

Todos os projetos foram desenvolvidos com **React Native + Expo**, utilizando o fluxo do :contentReference[oaicite:0]{index=0} para testes em dispositivos móveis.

---

## 🚀 Objetivo do repositório

Este repositório tem como objetivo documentar o aprendizado e desenvolvimento de aplicações mobile durante a disciplina, explorando conceitos como:

- Estruturação de aplicativos mobile
- Componentes em React Native
- Navegação entre telas
- Uso de câmera e notificações
- Interação com APIs do Expo
- Testes em dispositivos reais via Expo Go

---

## 📂 Projetos incluídos

### 🧮 CalculadoraApp
Aplicativo de calculadora simples desenvolvido em React Native.

**Funcionalidades:**
- Operações matemáticas básicas (soma, subtração, multiplicação, divisão)
- Interface simples e responsiva
- Lógica de estado com React Hooks

---

### ⚠️ ClickAlerta App
Aplicativo com foco em interação e recursos do dispositivo.

**Funcionalidades:**
- Uso da câmera do celular
- Captura de imagens
- Sistema de alertas/notificações
- Interface interativa
