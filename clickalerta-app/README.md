# 📱 ClickAlerta App

O ClickAlerta App é um aplicativo mobile desenvolvido com React Native + Expo que permite tirar fotos com a câmera do dispositivo e receber notificações locais após cada captura.

---

# 🎯 Objetivo do projeto

Este projeto foi criado para praticar:

- 📷 Uso da câmera no React Native (Expo Camera)
- 🔔 Notificações locais (Expo Notifications)
- 🧠 Manipulação de estado com React
- 📱 Desenvolvimento de aplicativos mobile com Expo
- ⚙️ Estruturação de um projeto real

---

# 🚀 Tecnologias utilizadas

- React Native
- Expo
- JavaScript / TypeScript
- expo-camera
- expo-notifications

---

# 📦 Requisitos

Antes de começar, você precisa ter instalado:

- Node.js
- npm ou yarn
- Aplicativo Expo Go no celular
